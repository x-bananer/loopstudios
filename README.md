# Loopstudios Website

Static landing page. Styled with Tailwind CSS. Reference: [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/loopstudios-landing-page-N88J5Onjw).

## Usage

Install dependencies

```
npm install
```

Run Tailwind CLI

```
npm run watch
```
